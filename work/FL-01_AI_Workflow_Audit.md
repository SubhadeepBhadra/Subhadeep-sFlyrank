# AI Workflow Audit (FL-01)

## Weekly Task Audit

| Task | Classification | Rationale |
|------|----------------|-----------|
| Watching AI/ML tutorials | Collaborate with AI | AI summarizes concepts, but I verify my understanding. |
| Writing Python code | Collaborate with AI | AI helps generate code, but I write, test, and understand it. |
| Debugging Python errors | Delegate to AI with review | AI suggests fixes, but I verify every solution before using it. |
| Learning Machine Learning concepts | Collaborate with AI | AI explains difficult topics with examples. |
| Writing GitHub README files | Delegate to AI with review | AI drafts documentation that I edit before publishing. |
| Preparing internship resume | Collaborate with AI | AI improves wording, but I ensure everything is truthful. |
| Writing Git commit messages | Fully automate | AI can generate concise commit messages automatically. |
| Planning my weekly study schedule | Delegate to AI with review | AI creates a schedule that I adjust based on my priorities. |
| Reading research papers | Collaborate with AI | AI summarizes papers, but I read important sections myself. |
| Applying for internships | Just Me | Final application decisions should be my own. |
| Taking online assessments | Just Me | Academic integrity requires that I complete them myself. |
| Writing professional emails | Delegate to AI with review | AI drafts emails, but I personalize and verify them. |

## My Three Target Tasks

| Task | Success Definition |
|------|--------------------|
| Debugging Python code | Fix the issue within 30 minutes and understand the cause of the error. |
| Writing README files | README includes installation, usage, features, and clear project description. |
| Learning ML concepts | I can explain the concept in my own words and implement a simple example. |
